Image Search Abstraction Layer microservice

Requirements

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings by using the /api/imagesearch/latestsearches/ endpoint.

Note - uses Giphy API for searching

Usage:

http://appurl/api/imagesearch/megatron
http://appurl/api/imagesearch/megatron?offset=2
http://appurl/api/imagesearch/latestsearches/
